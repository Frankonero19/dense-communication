# Dense Communication

Cursor AI skill that maximizes information density in LLM responses and reasoning.

## Principles

- **BLUF** — conclusion first
- **Orwell compression** — cut filler
- **Inverted pyramid** — order by importance
- **Meaning-ink ratio** — every token carries data
- **Feynman checks** — simplify to verify understanding

## Files

| File | Purpose |
|------|---------|
| `SKILL.md` | Skill definition — core rules, output modes, self-check |
| `anti-patterns.md` | Kill list of filler, hedging, and structural anti-patterns |

## Usage

Place the skill folder into `~/.cursor/skills/dense-communication/` and it will be available to the Cursor agent automatically.
